Flexbox

Class name Values

l-flex Make the container a flex container
u-row Arrange flex items horizontally
u-col Arrange flex items vertically
u-wrap Allow flex items to wrap onto multiple lines
u-nowrap Prevent flex items from wrapping
u-wrap-reverse Reverse the wrapping direction
u-justify-start Align flex items to the start of the flex container
u-justify-end Align flex items to the end of the flex container
u-justify-center Center align flex items along the main axis
u-justify-between Distribute flex items evenly with space between
u-justify-around Distribute flex items evenly with space around
u-align-start Align flex items to the start of the cross axis
u-align-end Align flex items to the end of the cross axis
u-align-center Center align flex items along the cross axis
u-align-baseline Align flex items to their baseline
u-align-stretch Stretch flex items to fill the container along the cross axis
u-align-start Align content to the start of the cross axis
u-align-end Align content to the end of the cross axis
u-align-center Center align content along the cross axis
u-align-between Distribute content evenly with space between
u-align-around Distribute content evenly with space around
u-align-stretch Stretch content to fill the container along the cross axis
u-align-self-auto Reset alignment for individual flex items
u-align-self-start Align individual flex items to the start of the cross axis
u-align-self-end Align individual flex items to the end of the cross axis
u-align-self-center Center align individual flex items along the cross axis
u-align-self-baseline Align individual flex items to their baseline
u-align-self-stretch Stretch individual flex items to fill the container along the cross axis
u-grow-0 Prevent flex items from growing
u-grow-1 Allow flex items to grow to fill available space
u-shrink-0 Prevent flex items from shrinking
u-shrink-1 Allow flex items to shrink if necessary
